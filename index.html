Contexto del Problema

Ahorrar una cantidad fija mensual puede volverse monótono o insuficiente para metas exponenciales. Este sistema implementa un método donde el hábito de ahorro se intensifica mes a mes siguiendo la curva natural de Fibonacci, permitiendo empezar con montos muy bajos y escalar de forma controlada.

Explicación del Algoritmo

El programa calcula el dinero correspondiente a cada mes sumando las bases de los dos meses anteriores ($A + B = C$). El algoritmo se ejecuta de manera limpia mediante variables simples de control numérico, prescindiendo totalmente de arrays o vectores para asegurar un procesamiento eficiente.

Conclusión

La sucesión de Fibonacci aplicada a las finanzas demuestra que la constancia y el incremento progresivo generan acumulaciones de capital sorprendentes en periodos cortos, adaptando dinámicas de crecimiento biológico a la economía doméstica.
